# Projeto: Top 5 melhores jogos (Atividade Acadêmica)

## 🚀 Sobre o projeto

Este projeto consiste em uma página web estática desenvolvida exclusivamente com HTML5, sem o uso de CSS ou JavaScript. O objetivo é demonstrar o domínio de tags semânticas, organização de listas, links internos/externos e estruturação de metadados, conforme os requisitos da atividade proposta.

O tema escolhido foi uma análise pessoal sobre os 5 melhores jogos de todos os tempos, apresentando critérios técnicos e nostalgia.

## 🛠️ TEcnologias utilizadas

* HTML5
